⬡ ¿Cómo usar EVMAudit?

EVMAudit analiza contratos inteligentes de Solidity combinando Slither, Mythril y Echidna, y genera un informe de vulnerabilidades con una puntuación de riesgo de 0 a 10.

1. Sube tu contrato

  • Pestaña FILE: arrastra o selecciona un archivo .sol.
  • Pestaña CODE: pega directamente el código Solidity y pon un nombre de contrato.

2. Espera el análisis

Verás una barra de progreso con las 7 fases del pipeline (Slither, Mythril, normalización, correlación, wrapper Echidna, fuzzing e informe). Puede tardar 1-2 minutos.

3. Revisa los resultados

  • Puntuación de riesgo global (0-10) y nivel (bajo/medio/alto).
  • Tabla de hallazgos correlacionados con severidad, confianza y estado (confirmado/detectado).
  • Resultados del fuzzing con Echidna por propiedad.

4. Descarga el informe

Con el botón "Descargar informe (JSON)" puedes guardar el resultado completo del análisis.
